# wap-mk-notes

本仓库用于归档和发布多个独立 HTML 页面。内容主要涉及一些技术笔记、工具页面或实验性前端演示，不针对任何具体网站或服务。

## 项目简介

`wap-mk-notes` 是一个静态页面归档仓库。每个 HTML 文件通常是一个独立的页面，包含完整的样式与脚本。仓库旨在提供一种轻量、直接的方式来保存和展示一些零散的前端片段或文档。

## 目录说明

- 仓库根目录下放置各 HTML 文件。
- 部分页面可能引用同一目录下的资源文件（如图片、CSS 或 JS），请确保文件路径相对正确。
- 暂未对文件按主题进行细分，后续可能根据数量调整结构。

## 页面归档说明

- 每个 HTML 页面均为独立完整内容，可直接在浏览器中打开查看。
- 部分页面可能包含外部资源引用（如 CDN 库），请确保网络连通。
- 页面可能随时间更新或移除，不保证长期可用性。

## 维护说明

- 本仓库由个人维护，更新频率不定。
- 欢迎通过 Issues 提交问题或建议。
- 原则上不接受直接向仓库提交的 Pull Request（除非是明显的错误修正）。

## 许可

本仓库内容仅供学习与参考。如有任何问题，请通过 Issues 联系。
